## 🚀 Installation
```bash
git clone https://github.com/denisidoro/dotfiles.git ~/.dotfiles
~/.dotfiles/bin/dot self install
```

## 🥳 Usage
```
dot
```

## 🔰 Updating
```bash
dot self update
```

## 🎯 Overriding configs

Edit the following files accordingly:

```sh
# shell
~/.dotfiles/local/zshrc

# git
~/.dotfiles/local/gitconfig
```
