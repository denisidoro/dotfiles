## 🚀 Installation
```bash
brew install denisidoro/tools/dotfiles
```

## ⌨️ Usage
```bash
dot <cmd> <args>... # dot rice pipes, for example
```

## 🌐 Updating
```bash
dot self update
```

## 🎯 Overriding configs

Edit the following files accordingly:

```sh
# shell
~/.dotfiles/local/zshrc

# git
~/.dotfiles/local/gitconfig
```
